获取路径： https://www.jsdelivr.com/package/gh/aixiu/cdn-assets

也可以直接拼接路径：

```
https://cdn.jsdelivr.net/gh/aixiu/cdn-assets/图片地址
```

例如：

```
https://cdn.jsdelivr.net/gh/aixiu/cdn-assets/placeholder/d570170f4f12.svg
```
